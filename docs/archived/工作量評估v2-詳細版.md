- 證書鏈需要 3 個憑證：Vendor Cert + Apple WWDR G3 中間 CA + Apple Root CA

## 風險與備註

1. **ngrok URL 變動**：免費版每次重啟 URL 會變，已裝入裝置的 MDM 描述檔會失效。生產環境需固定域名。
2. **憑證續期**：APNS 推播憑證、Vendor Certificate、DEP Token 均為 1 年有效期，需定期續期。
3. **APNS 簽署格式**：Apple Push Certificates Portal 要求特定的 base64 plist 格式（非標準 PKCS#7），已適配並驗證通過。
