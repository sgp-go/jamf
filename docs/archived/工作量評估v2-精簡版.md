# 平板管理專案（第二期）— 工作量評估

## 專案目標

建立自建 MDM 伺服器，將裝置從 Jamf Pro 遷出，透過 ABM + ADE 實現裝置自動註冊到自建 MDM，完全自主管控裝置管理能力。
