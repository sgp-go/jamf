# 臺南平板管理測試專案 — 工作量評估

## 專案目標

讓平板管理平臺連線 Jamf，透過 Jamf 推送安裝 Agent App 到 iPad，並透過 Jamf API 和 Agent App 雙通道獲取裝置狀態。

## 風險與假設

1. **DeviceGuardKit**：FamilyControls 授權和使用時長採集需在真機上充分驗證穩定性。
2. **伺服器部署**：後端當前為本地開發模式，生產部署（伺服器、域名、HTTPS）需額外協調。
